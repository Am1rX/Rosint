## Getting Started

This project is created by @str4n5er due to university project which is can be used for personal use, follow the installation.

### Installation

1. Clone the repo
   ```sh
   pip install -r requirements.txt
   ```
2. Install NPM packages
   ```sh
   python rosint.py
   ```
